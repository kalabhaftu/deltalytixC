## Widget Sizes Available
- `tiny`: Small compact widgets
- `small`: Standard small size
- `small-long`: Small but elongated
- `medium`: Standard medium size
- `large`: Large widgets
- `extra-large`: Maximum size widgets
- `kpi`: Special KPI widget size

## Categories
- **charts**: Data visualization widgets
- **statistics**: Statistical summary widgets  
- **other**: Special widgets like calendar
